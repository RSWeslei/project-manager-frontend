### Frontend

1.  Em um novo terminal, navegue até a pasta do frontend:
    ```bash
    cd project-manager-frontend
    ```
2.  Instale as dependências:
    ```bash
    npm install
    ```
3.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```

- A aplicação estará disponível em `http://localhost:5173`.
